Konfiguro Hurma AI

Për të biseduar me AI-in, na duhet çelësi yt API nga Anthropic. Ai ruhet vetëm në browser-in tënd dhe nuk dërgohet askund tjetër veç tek Anthropic.

  1. Shko te console.anthropic.com
  2. Klikoji Create Key
  3. Kopjoje çelësin (fillon me sk-ant-...)
  4. Ngjite poshtë

💡 Privatësi: Çelësi mbetet vetëm në pajisjen tënde (localStorage). Të dhënat e dyqanit dërgohen direkt tek Anthropic API për përgjigje, jo tek ndonjë server tjetër.

💰 Kosto: ~$0.003 për pyetje (1 ден). 100 pyetje = ~30 ден.

Po ngarkohet…

Mirë se erdhe! Unë jam Hurma AI 🌴 — asistenti yt personal për dyqanin.

Unë e njoh dyqanin tënd: shitjet, klientët, stokun, borxhet, fitimin. Mund të më pyesësh çdo gjë në shqip, p.sh.:

  • "Sa borxh ka Sulejmani?"
  • "Cili produkt po mbaron stokun?"
  • "Sa kam shitur këtë muaj?"
  • "Më rekomando një strategji çmimi"

Provo një nga pyetjet e shpejta lart, ose shkruaj direkt poshtë.
